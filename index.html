Règles de Confidentialité

Bienvenue dans l'application Le Coran pour Tous. Votre confidentialité est très importante pour nous. Cette page décrit comment nous collectons, utilisons et protégeons vos informations.

1. Données collectées

Nous collectons uniquement les données nécessaires pour fournir les services de l'application, notamment :

  • Votre position géographique pour déterminer la Qibla et les horaires de prière.
  • Aucune donnée personnelle identifiable n'est collectée à d'autres fins.

2. Utilisation des données

Les données collectées sont utilisées uniquement pour :

  • Afficher la direction de la Qibla.
  • Calculer les horaires de prière en fonction de votre position géographique.
Ces données ne sont jamais partagées avec des tiers.

3. Sécurité

Nous mettons en œuvre des mesures techniques et organisationnelles appropriées pour protéger vos données contre tout accès non autorisé, modification, divulgation ou destruction.

4. Droits des utilisateurs

Vous avez le droit de :

  • Demander des informations sur les données collectées.
  • Demander la suppression des données collectées.

5. Contact

Si vous avez des questions concernant cette politique de confidentialité, vous pouvez nous contacter à : support@lecoranpourtous.com.
